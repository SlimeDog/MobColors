# MobColors
Manage mob colors and variants, on a per world basis 

**MobColors** manages mob colors and variants. The following mob-types may be managed:
- axolotl
- cat
- fox
- horse
- llama
- mooshroom
- rabbit
- sheep
- shulker
- tropical_fish
