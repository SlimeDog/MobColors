### Overview

**MobColors** manages mob colors and variants, on a per-world basis. The following mob-types may be managed:<br>
&mdash; axolotl<br>
&mdash; cat<br>
&mdash; fox<br>
&mdash; frog<br>
&mdash; horse<br>
&mdash; llama<br>
&mdash; mooshroom<br>
&mdash; parrot<br>
&mdash; rabbit<br>
&mdash; sheep<br>
&mdash; shulker<br>
&mdash; tropical_fish<br>
