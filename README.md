# MobColors
Manage colors of mobs, per world 

**MobColors** manages the colors of mobs that present in different colors or variants. The following mob-types may be managed:
- axolotl
- cat
- fox
- horse
- llama
- mooshroom
- rabbit
- sheep
- shulker
- tropical_fish
